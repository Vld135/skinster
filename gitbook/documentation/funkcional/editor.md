# Overview

<figure><img src="../.gitbook/assets/overview.png" alt=""><figcaption></figcaption></figure>

На данной странице у вас есть возможность просматривать все предметы которые есть у вас в инвентаре (Включая предметы из хранилищ)

Для каждого предмета можно посмотреть цену, качество, float, статус protected а так же наклейки и брелки если таковы есть на предмете

Возле меню выбора хранилища так же отображается сумма всего инвентаря указанная в валюте аккаунта стим (Поскольку цены конвертируются из USD то цена может не соответствовать цене на ТП)
