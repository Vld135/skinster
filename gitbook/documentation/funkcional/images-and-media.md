# In-game shop

<figure><img src="../.gitbook/assets/ingame.png" alt=""><figcaption><p>Add alt text and captions to your images</p></figcaption></figure>

Данный раздел позволяет купить любой предмет из игрового магазина, будет полезно для тех кто фармит Armory или кто закупает капсулы мажоров\
Нет надобности сидеть и мучаться с покупкой по 20 капсул через игру, просто укажи количество и нажми кнопку купить

> Steam позволяет без проблем купить \~1600-1800 капсул за короткий промежуток времени. Затем он накладывает временное ограничение на 2-3 часа
