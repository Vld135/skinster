# Trade Offers

<figure><img src="../.gitbook/assets/tradeoffers.png" alt=""><figcaption></figcaption></figure>

По умолчанию доступно два инвентаря CS2 и Steam\
При надобности добавить инвентарь другой игры

* Нажми кнопку Settings&#x20;
* Затем Добавить инвентарь (Add inventory)
* Укажи Id игры + контекст инвентаря. Например: CS2 - 730/2

P2P - Позволяет создавать обмены между выбранным аккаунтом и аккаунтом чей trade link вы укажите

Looter - Передает все предметы из инвентарей что указаны в настройках. Позволяет выбрать несколько аккаунтов и скинуть весь их дроп на определенный trade link
