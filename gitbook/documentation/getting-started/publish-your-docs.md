# Системные требования

Данный раздел больше предназначен для людей кто будет запускать автофарм

Рекомендованные системные требования

* Процессор - AMD Ryzen 5600X или Intel i5-10400f

* Оперативная память - 32 GB
* SSD - 50 GB свободного места
